Taxi Native

Taxi Native es una aplicación para teléfonos Android e IOS que permite conectar a conductores con pasajeros. La aplicación ha sido desarrollada utilizando react native, react-native-maps, redux y react-navigation. Además, la aplicación utiliza un servidor desarrollado en nodejs y que se encarga de manejar las peticiones de conductores y pasajeros en tiempo real.
